# 📱 KDailyUtil

**데일리 유틸리티 안드로이드 애플리케이션**  
일상의 편의를 위해 뉴스 브리핑, 운전 중 말하기 연습, 오디오 캡처 등 실용적인 기능을 제공합니다.

---

## ✨ 주요 기능

### 📰 뉴스 키워드 브리핑 (Morning Briefing)
- 사용자가 설정한 관심사/키워드별 국내 최신 뉴스 수집.
- **AI 요약**: Gemini 1.5 Flash 엔진을 통해 기사를 자연스러운 대화체로 요약.
- **TTS 낭독**: 수집된 뉴스 요약을 음성으로 들려주어 바쁜 아침 시간에 시각적 자유 제공.

### 🎙 오디오 시스템 기능 및 최적화 (Updated)
*   **재생목록(Playlist) 및 폴더 관리**:
    *   **폴더 기반 관리**: `Download/KDailyUtil` 하위에 폴더를 생성하여 오디오 파일을 재생목록별로 분류 관리.
    *   **파일 이동 기능**: 선택한 파일을 원하는 재생목록으로 자유롭게 이동.
*   **하이브리드 녹음 모드**: 
    *   **내부 소리**: 시스템 미디어 소리를 고음질로 캡처 (MediaProjection).
    *   **외부 마이크**: 사용자 목소리를 직접 녹음 (AudioSource.MIC).
*   **파일 접근성 및 관리**:
    *   **공용 폴더 저장**: `Download/KDailyUtil` 폴더에 저장되어 폰의 '내 파일' 앱에서 직접 복사/공유/삭제 가능.
    *   **뉴스 쉐도잉 전용 관리**: 쉐도잉 연습 녹음은 `뉴스_쉐도잉` 서브 폴더에 별도로 보관되어 식별 용이.
    *   **휴지통(Trash) 시스템**: 실수로 삭제한 파일을 복구할 수 있는 휴지통 기능 제공.
    *   **숨김 파일 관리**: 리스트에서 보이고 싶지 않은 파일을 별도로 숨기고 관리.
*   **사용자 중심 UI/UX**:
    *   **리스트 순서 변경**: 드래그 앤 드롭 방식의 위/아래 이동 및 순서 영구 저장.
    *   **화면 꺼짐 및 백그라운드 재생 안정화**: `MediaSession` 및 `WakeLock` 적용으로 어떠한 환경에서도 끊김 없는 재생 보장.
    *   **편집 잠금(Edit Lock)**: 데이터 유실 방지를 위한 보안 장치.

### 🚗 뉴스 쉐도잉 (News Shadowing) - Formerly Driving Shadowing
- **고급 사설/칼럼 중심**: 학습 가치가 높은 오피니언 리더들의 글을 본문까지 통째로 추출하여 학습.
- **쉐도잉 반복 루프**: 
    1.  **TTS 낭독**: 먼저 정확한 발음으로 듣기.
    2.  **대기 및 녹음**: 따라 말하는 동안 목소리를 자동 녹음.
    3.  **검토**: 연습 음성을 즉시 다시 들으며 발음 교정.
- **운전 안전 UI**: 고대비 블랙 테마와 대형 버튼으로 조작 편리성 극대화.
- **안정적인 녹음 관리**: 기기 설정에 구애받지 않는 강건한 저장 시스템 및 자동 권한 획득 구현.

## 🛠 기술 스택

- **UI**: Jetpack Compose (Modern & Reactive UI)
- **Networking**: Jsoup (RSS Scraping)
- **AI**: Google Generative AI SDK (Gemini 1.5 Flash)
- **Concurrency**: Kotlin Coroutines & Flow
- **Data Persistence**: Preferences DataStore
- **Audio**: TextToSpeech (TTS), MediaRecorder, MediaPlayer

## 🚀 시작하기

1. **저장소 클론**
   ```bash
   git clone https://github.com/your-repo/KDailyUtil.git
   ```

2. **환경 설정 (Gemini API)**
   - [Google AI Studio](https://aistudio.google.com/app/apikey)에서 API 키를 발급받습니다.
   - 앱 내 **설정(Morning Settings)** 화면에서 API 키를 입력하면 AI 요약 기능이 활성화됩니다.
   - API 키가 없는 경우 제목만 나열하는 **데모 모드**가 작동합니다.

3. **빌드 및 실행**
   - Android Studio (Ladybug 이상)에서 프로젝트 오픈.
   - `./gradlew assembleDebug` 코드로 빌드 후 실행.

## 🛠 최근 업데이트 및 작업 현황 (2026-04-23)

### 🚀 뉴스 추출 파이프라인 고도화
특정 매체(한겨레, 조선일보 등)의 구글 뉴스 RSS 리다이렉트 및 안티 크롤링 문제를 해결하기 위해 **3단계 하이브리드 추출 시스템**을 구축하고 본문 정제 로직을 안착시켰습니다.

1.  **Tier 1 (Protobuf Decoder)**: 구글 뉴스 고유의 바이너리 인코딩(`CBMi...`)을 분석하여 실제 매체 URL을 즉시 해독하는 전용 디코더 구현. (2025년 최신 구조 대응)
2.  **Tier 2 (Jsoup Redirect & Meta Refresh)**: 표준 HTTP 리다이렉트와 HTML 내 `<meta http-equiv="refresh">` 태그를 동시에 추적하여 리다이렉트 성공률 극대화.
3.  **Tier 3 (WebView Fallback)**: 자바스크립트 실행이 필수적인 사이트를 위해 백그라운드 WebView를 구동, 최종 랜딩 페이지를 가로채는 방식 적용.

### 📝 본문 정밀 정화 시스템
- **노이즈 제거**: 사진 설명(figcaption), 부제목, 기자 정보, 오디오 플레이어 문구 등 기사 외 요소를 완벽 제거.
- **다중 블록 결합**: 중간에 광고가 삽입되어 끊긴 본문도 누락 없이 하나로 통합 추출 (5,000자 상향 지원).
- **바이라인 청소**: 기사 끝부분의 이메일, 특파원/기자 정보, 잔여 기호 등을 깔끔하게 정리.
