# 📱 KDailyUtil

**데일리 유틸리티 안드로이드 애플리케이션**  
일상의 편의를 위해 뉴스 브리핑, 운전 중 말하기 연습, 오디오 캡처 등 실용적인 기능을 제공합니다.

---

## ✨ 주요 기능

### 📰 뉴스 키워드 브리핑 (Morning Briefing)
- 사용자가 설정한 관심사/키워드별 국내 최신 뉴스 수집.
- **AI 요약**: Gemini 1.5 Flash 엔진을 통해 기사를 자연스러운 대화체로 요약.
- **TTS 낭독**: 수집된 뉴스 요약을 음성으로 들려주어 바쁜 아침 시간에 시각적 자유 제공.

### 2. 오디오 캡처 및 관리 (Audio Capture & Management)
- **고성능 로딩**: 대량의 파일도 지연 없이 불러오는 최적화된 데이터 로딩 시스템.
- **혁신적 UI/UX**:
    - **통합형 플레이어**: 별도의 재생 탭 없이 어디서든 제어 가능한 **지속형 미니 플레이어** 도입.
    - **확장형 컨트롤**: 미니 플레이어를 확장하여 상세 조작이 가능한 **바텀 시트 플레이어**.
    - **탭 간소화**: 녹음, 파일, 목록 중심의 직관적인 3개 탭 구성.
- **스마트 파일 관리**:
    - **중복 자동 제거**: 표준 경로(Canonical Path) 기반의 정확한 파일 중복 체크.
    - **상세 정보 확인**: 파일의 전체 경로, 크기, 생성일을 확인할 수 있는 상세 정보 다이얼로그.
    - **안전한 삭제**: 휴지통 기능을 포함한 안정적인 파일 이동 및 복구 로직.
    - **가져오기/숨기기**: 외부 파일 가져오기 및 보안을 위한 파일 숨김 기능.
- **고급 설정**: 비트레이트, 샘플링 레이트, 녹음 소스(마이크/내부 오디오) 커스텀 설정.
- **지능형 재생 제어**:
    - **블루투스 대응**: 블루투스 이어폰이나 스피커 연결이 해제되면 자동으로 재생을 일시정지(`NoisyAudioReceiver`)하여 의도치 않은 외부 출력을 방지.
    - **반복 재생 로직 최적화**: 재생목록 내 순차 재생, 전체 반복, 셔플 모드 버그 수정 및 안정화.
- **파일 및 저장소 보안 최적화**:
    - **Scoped Storage 완벽 대응**: 안드로이드 11+ 보안 정책에 맞춰 재생목록 데이터는 **내부 저장소**에, 녹음 파일은 **앱 전용 외부 저장소**(`Android/data/.../files`)에 안전하게 보관.
    - **자동 마이그레이션**: 이전 버전의 데이터를 감지하여 새로운 통합 폴더로 안전하게 자동 이동.
    - **외부 목록 임포트**: `.plt` 파일을 직접 선택하여 재생목록으로 즉시 등록 가능 (Toast 피드백 지원).
- **비간섭 플로팅 컨트롤러**:
    - 녹음 준비 상태에서 앱을 사용 중일 때는 UI를 가리지 않으며, **백그라운드로 전환될 때만** 플로팅 아이콘이 나타나도록 라이프사이클 최적화.

### 🚗 뉴스 쉐도잉 (News Shadowing)
- **고급 사설/칼럼 중심**: 학습 가치가 높은 오피니언 리더들의 글을 본문까지 통째로 추출하여 학습.
- **쉐도잉 반복 루프**: TTS 낭독 -> 대기 및 녹음 -> 연습 음성 즉시 재청취 루프 구현.
- **운전 안전 UI**: 고대비 블랙 테마와 대형 버튼으로 조작 편리성 극대화.

## 🛠 기술 스택

- **UI**: Jetpack Compose (Modern & Responsive UI)
- **Networking**: Jsoup (RSS Scraping)
- **AI**: Google Generative AI SDK (Gemini 1.5 Flash)
- **Concurrency**: Kotlin Coroutines & Flow
- **Data Persistence**: Preferences DataStore
- **Audio**: TextToSpeech (TTS), MediaRecorder, MediaPlayer

## 🚀 시작하기

1. **저장소 클론**
   ```bash
   git clone https://github.com/your-repo/KDailyUtil.git
   ```

2. **환경 설정 (Gemini API)**
   - [Google AI Studio](https://aistudio.google.com/app/apikey)에서 API 키를 발급받습니다.
   - 앱 내 **설정(Morning Settings)** 화면에서 API 키를 입력하면 AI 요약 기능이 활성화됩니다.
   - API 키가 없는 경우 제목만 나열하는 **데모 모드**가 작동합니다.

3. **빌드 및 실행**
   - Android Studio (Ladybug 이상)에서 프로젝트 오픈.
   - `./gradlew assembleDebug` 코드로 빌드 후 실행.

### 🚀 오디오 엔진 및 UX 전면 개편
